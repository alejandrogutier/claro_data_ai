# Prompt: Analysis v1

## Objetivo
Generar sintesis agregada de un conjunto de clasificaciones para identificar narrativas, riesgos y oportunidades.

## Reglas
- Responder exclusivamente en JSON valido.
- Basar conclusiones en la evidencia disponible.
- Incluir nivel de confianza por insight agregado.

```json
{
  "sintesis_general": "string",
  "narrativa_principal": "string",
  "oportunidades_negocio": ["string"],
  "riesgos_reputacionales": ["string"],
  "temas_dominantes": ["string"],
  "nivel_confianza": 0.0
}
```
