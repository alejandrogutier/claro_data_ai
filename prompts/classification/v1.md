# Prompt: Classification v1

## Objetivo
Clasificar cada contenido en una taxonomia operativa para monitoreo y analisis.

## Reglas
- Responder exclusivamente en JSON valido.
- No inventar datos que no aparezcan en el contenido.
- Si falta informacion, usar valores `null` y confianza baja.

## Output esperado
```json
{
  "categoria": "string",
  "sentimiento": "positivo|neutro|negativo",
  "etiquetas": ["string"],
  "confianza": 0.0,
  "resumen": "string"
}
```

## Contexto
- source_type: {{source_type}}
- provider: {{provider}}
- language: {{language}}
- titulo: {{title}}
- resumen: {{summary}}
- contenido: {{content}}
